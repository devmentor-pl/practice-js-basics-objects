# `#01` JavaScript: Podstawy – obiekty

Utwórz obiekt, który będzie przypisany do zmiennej `user`. 

Obiekt ten ma posiadać 4 właściwości:
* firstName,
* lastName,
* sex,
* age.

Następnie wartość każdej właściwości obiektu wyświetl w konsoli.
